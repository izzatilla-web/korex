# Korex

Tozalangan Next.js 16 App Router starter. Asosiy sahifa SSR uchun request vaqtida render bo'ladi.

## Ishga tushirish

```bash
npm run dev
```

## Muhim fayllar

- `app/page.tsx` - asosiy server-rendered sahifa
- `app/layout.tsx` - root layout va metadata
- `app/globals.css` - global stillar

## Buyruqlar

```bash
npm run dev
npm run build
npm run start
npm run lint
```
